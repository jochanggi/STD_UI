구조 레이아웃

영역구분
구분 예약어 설명
바깥 영역 -wrap UI 또는 컨텐츠의 전체영역
지역 영역 -area 레이아웃의 영역을 분리한다.
그룹 영역 -group 연속적인 인라인 엘리먼트를 그룹화한다. (예) .btn-group, .label-group, .form-group, .ico-group
안쪽 영역 -inner 내부영역 2중 처리

구조 의미

영역구분
구분 예약어 설명
머리글 영역 -head 전체영역의 목차포함
본문 영역 -body 전체영역의 본문
바닥글 영역 -foot 전체영역의 바닥글
제목 영역 -title 전체내용 중 일부의 제목
본문 영역 -cont 전체내용 중 일부의 내용

메뉴 -menu 탐색 역할이 없는 선택목록 메뉴 (예) .drop-menu, .select-menu
목록 -list ul, ol 에 해당하는 반복되는 목록그룹
항목 -item 리스트 항목
링크 -link 링크 항목
토글버튼 -toggle 토글 기능을 갖는 버튼
열기버튼 -open 열기 기능을 갖는 버튼
닫기버튼 -close 닫기 기능을 갖는 버튼
버튼 텍스트 -label 텍스트만 별도로 마크업하는 경우

레이아웃 유형

레이아웃
구분 예약어 설명
그리드 기본 .grid Floating 레이아웃 방식
그리드 균등분할 .grid-flex Flexible 레이아웃 방식
그리드 균등분할 .grid-table Table 레이아웃 방식
레이아웃 유연하게 grid-fluid 너비 초기화 (기본너비, 최소너비, 최대너비)
